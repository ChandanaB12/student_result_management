# 🎓 Student Result Management System

A web-based Student Result Management System developed using Django.

## Features

- Student Login using USN
- Admin Dashboard
- Add Students
- Add Results
- Subject Code
- Subject Name
- Internal Marks
- External Marks
- Total Marks
- Percentage Calculation
- Grade Calculation
- PASS / FAIL Status
- Semester
- Academic Year
- Print Result

## Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite

## Project Structure

- Student Management
- Result Management
- Admin Dashboard
- Student Result Portal
